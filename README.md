# OHO Boot Module

项目依赖：`Java 8` `Spring Boot 2.7.6`

## 项目结构

```txt
oho-boot-module
│          
├─oho-common		# 公共模块
│ 
├─oho-dependencies	# 依赖版本管理
│      
├─oho-elasticsearch	# elasticsearch工具模块
│                          
├─oho-mail		# mail 模块
│                          
├─oho-mybatis		# mybatis工具模块
│                          
├─oho-oss		# OSS工具模块
│                          
├─oho-payment		# 支付工具模块
│                          
├─oho-quartz		# 定时任务工具模块模块
│                          
├─oho-redis		# Redis工具模块
│                          
└─oho-rocketmq		# RocketMQ工具模块
```
